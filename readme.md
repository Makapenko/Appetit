Приложение "Аппетитка"

Сервис для формирования, хранения и отправки заказов, 
заказная разработка для производителя соусов и солений.

Используемые технологии:

Frontend:
- HTML
- CSS
- React
- Redux
- PWA

Backend:
- NodeJS
- Express
- MongoDB
- Mongoose

Ссылка на сайт: http://appetitka.herokuapp.com/
